### 模版语句
> 用来响应绑定目标的触发的事件
+ 模版语句不能引用全局命名空间的任何东西。比如window、document、console.log（）、Math.max

`(en=vent)='statement'`
#### 语句上下文
> 通常是正在绑定事件的那个组件实例

#
### 模版引用变量 (#var)
> 使用井号（#）声明模板引用变量。
#
### 数据绑定与HTML
例子
```HTML
<button [disabled]="isUnchanged">Save</button>
```
+ 注意这里绑定到的是DOM元素disable的property而不是attribute
